# Payal Fofadiya - Portfolio Website

A personal portfolio website for Payal Fofadiya, AI Agent Systems Leader at LinkedIn. Built with Next.js 15, React 19, Tailwind CSS v4, and Motion animations.

## Features

- Minimal one-page portfolio layout
- Blog support with MDX
- Responsive and accessible design
- Smooth animations powered by Motion
- Dark/light theme support

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack

- **Framework**: Next.js 15 with App Router
- **Styling**: Tailwind CSS v4
- **Components**: React 19 with TypeScript
- **Animations**: Motion library
- **Content**: MDX for blog posts

## Deployment

This project is configured for Vercel deployment with automatic builds.
